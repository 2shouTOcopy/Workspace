# 会议与办公

## 1. 会议纪要整理

```text
请把当前内容整理成会议纪要，输出结构如下：
1. 会议主题
2. 关键结论
3. 待办事项
4. 责任人
5. 截止时间
如果缺失，请标注待补充。
```

## 2. 待办提取

```text
请从当前内容中提取所有可执行事项，并输出为 Markdown checklist。
要求：合并重复项；不明确事项补一句说明。
```

## 3. 周报初稿

```text
请根据当前内容，生成一版周报初稿，结构如下：
1. 本周完成事项
2. 本周亮点
3. 当前问题与风险
4. 下周计划
要求：正式、简洁、不要夸张。
```

## 4. 日报整理

```text
请根据当前内容，整理为一版日报，输出：今日完成、遇到问题、明日计划。
要求：每部分控制在 3 条以内。
```

## 5. 任务优先级梳理

```text
请根据当前内容，整理任务优先级。
输出：
1. 高优先级
2. 中优先级
3. 低优先级
4. 建议先做的三件事
如果依据不足，请明确说明。
```

## 6. 邮件/消息正式化

```text
请把当前内容改写成适合发给同事或客户的正式表达。
要求：礼貌、清晰、简洁，输出最终版本，不要解释。
```
