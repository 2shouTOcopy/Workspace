# 写作与改写

## 1. 正式表达改写

```text
请把当前内容改写成更正式、专业的表达。
要求：保留原意；删除口语化表达；句子更简洁；输出最终版本。
```

## 2. 压缩成短版

```text
请把当前内容压缩到 150 字以内。
要求：保留核心信息；删除重复表述；输出最终版本。
```

## 3. 扩展成完整段落

```text
请把当前要点扩展成一段完整、自然、可直接使用的说明文字。
要求：逻辑清楚，不要空话。
```

## 4. 改成条目结构

```text
请把当前内容改写成结构化条目。
输出格式：
1. 背景
2. 核心观点
3. 风险
4. 建议
```

## 5. 改成汇报口径

```text
请把当前内容改写成适合向上汇报的表达。
要求：重点前置、语言克制、保留事实、不夸张。
```

## 6. 检查歧义

```text
请审阅当前内容，指出：
1. 不清楚的表达
2. 可能被误解的地方
3. 建议如何改写
要求：直接指出问题，不要泛泛表扬。
```
