# 知识管理

## 1. 当前笔记总结

```text
请基于当前笔记，输出：
1. 一句话总结
2. 三个核心观点
3. 两个待深入问题
要求：提炼而不是复述。
```

## 2. 标签建议

```text
请根据当前笔记内容，给出建议标签。
要求：优先长期可复用标签；不要太泛；控制在 3 到 5 个。
```

## 3. 双链建议

```text
请根据当前笔记内容，给出建议双链主题，并说明每个双链的理由。
```

## 4. 原子笔记拆分

```text
请把当前笔记拆成适合长期积累的原子化知识点。
输出：建议拆分主题、每个主题标题、保留内容、建议双链。
```

## 5. 主题归类

```text
请判断当前笔记更适合归入哪个知识主题，并给出理由。
如果存在多个主题，请按主次排序。
```

## 6. 知识库问答

```text
请基于我的 Obsidian 笔记回答这个问题：[问题]
如果现有笔记无法支持结论，请明确说“信息不足”。
```
