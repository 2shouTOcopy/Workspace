# 提示词模板库使用说明

这个目录里的模板，不是固定咒语，而是可复用骨架。

## 1. 每个模板怎么改

每次主要改 3 个部分：

1. 输入范围
- 当前笔记
- 当前选中文本
- 我的 vault 笔记

2. 输出格式
- 列表
- checklist
- 固定标题
- 表格

3. 约束条件
- 简洁
- 正式
- 不编造
- 信息不足要说明

## 2. 通用骨架

```text
请基于[输入范围]，完成[任务动作]。
输出格式：[输出格式]
约束： [约束条件]
```

## 3. 推荐默认约束

```text
如果信息不足，请明确指出；不要补造事实；语言简洁；输出可直接复用。
```

## 4. 使用建议

1. 先短提示，再追问
2. 先整理，再改写
3. 先结构化，再判断

如果一个模板效果一般，不要立刻换模型，先改输入范围和输出格式。
