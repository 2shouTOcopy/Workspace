# 分析与复盘

## 1. 复盘整理

```text
请把当前内容整理成复盘，结构如下：
1. 目标
2. 实际结果
3. 做得好的地方
4. 问题与原因
5. 后续优化动作
要求：具体，不要空泛。
```

## 2. 风险识别

```text
请根据当前内容，识别潜在风险。
输出：
1. 风险点
2. 触发条件
3. 影响
4. 建议应对方式
如果依据不足，请明确说明。
```

## 3. 决策梳理

```text
请根据当前内容，整理出：
1. 当前结论
2. 支撑依据
3. 不确定点
4. 下一步建议
```

## 4. 问题定位

```text
请阅读当前内容，指出：
1. 已知问题
2. 可能原因
3. 还缺哪些信息
4. 最值得先验证的点
```

## 5. 方案对比

```text
请根据当前内容，对可选方案进行比较。
输出：
1. 方案 A
2. 方案 B
3. 各自优缺点
4. 推荐方案与理由
如果材料不足，请明确说明。
```

## 6. 信息缺口识别

```text
请审阅当前内容，指出还缺哪些关键信息会影响判断。
输出按重要性排序。
```
